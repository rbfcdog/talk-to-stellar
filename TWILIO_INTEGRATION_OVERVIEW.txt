═══════════════════════════════════════════════════════════════════════════════════════

  ✅ Receives messages from Twilio when users send WhatsApp messages
  ✅ Forwards messages to your TalkToStellar AI agent

  ✅ Validates Twilio signatures for security
  ✅ Handles errors gracefully

🚀 QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════════════

1️⃣  Install Twilio SDK
    cd backend && npm install

2️⃣  Copy environment config
    cp .env.example .env
    
    Then edit .env and add:
    TWILIO_ACCOUNT_SID=your_account_sid_from_twilio
    TWILIO_AUTH_TOKEN=your_auth_token_from_twilio

3️⃣  Start your services
    
    Terminal 1 (Backend):
    npm run dev
    
    Terminal 2 (Python Agent):
    cd talktostellar-agent && python main.py
    
    Terminal 3 (Test):
    cd backend && ./test-webhook.sh

4️⃣  Get your public URL
    
    Using ngrok (free & easy):
    npm install -g ngrok
    ngrok http 3000
    
    Copy the URL: https://xxxx-xx-xxxxx-xxx.ngrok.io

5️⃣  Configure in Twilio Console
    
    1. Go to https://www.twilio.com/console
    2. Navigate to Messaging → Services → WhatsApp
    3. Set Webhook URL: https://your-url/agent-webhook/message
    4. Click Save
    
    ✓ DONE! Now WhatsApp messages go to your agent!

🔐 SECURITY FEATURES
═══════════════════════════════════════════════════════════════════════════════════════

✅ Twilio Signature Validation
   Every webhook request is validated with HMAC-SHA1
   Ensures messages really come from Twilio

✅ Environment-based Configuration
   Credentials stored in .env (never in code)
   Different for dev/prod environments

✅ Error Handling
   Graceful error responses
   No sensitive data leaks
   Structured error messages

✅ Type Safety
   Full TypeScript support
   Prevents runtime errors

🎓 ARCHITECTURE AT A GLANCE
═══════════════════════════════════════════════════════════════════════════════════════

TwilioController
  ├─ Receives HTTP requests from Twilio
  ├─ Validates format
  └─ Processes asynchronously & sends replies

TwilioService
  ├─ Communicates with Python agent
  ├─ Manages session IDs
  ├─ Formats responses for Twilio
  └─ Handles agent errors

TwilioValidationMiddleware
  ├─ Validates Twilio signatures (HMAC-SHA1)
  ├─ Prevents unauthorized requests
  └─ Optional in dev, required in prod

TwilioRouter
  ├─ Defines routes
  ├─ Applies middleware
  └─ Maps to controllers

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════════════

Problem: "Agent not responding"
→ Check Python agent is running on port 8000
  cd talktostellar-agent && python main.py

Problem: "Invalid Twilio signature"
→ Verify TWILIO_AUTH_TOKEN in .env is correct
→ Check it matches your Twilio Console token

Problem: "Webhook not receiving messages"
→ Verify public URL and Twilio webhook configuration
→ Test with test-webhook.sh first

Problem: "Message not delivered to user"
→ Check WhatsApp number is verified in Twilio
→ Verify webhook URL is correct in Twilio console
